# TASS TechAI v1

This is the mobile-optimized version of the **TASS TechAI** application.  
It includes:
- Login Screen
- Modern Dashboard (Efficiency, Resources, Diagnostics, Chat, Admin)
- AI Chat Assistant Panel
- Responsive Blue-White UI Theme

### Deployment Steps:
1. Extract this ZIP file.
2. Push extracted files to a new GitHub repository.
3. Deploy on [Vercel](https://vercel.com/).
